# ReactJS Mini Portfolio

## Deskripsi
Mini project ini dibuat untuk latihan ReactJS dari 0.  
Fitur:
- Tampilkan daftar teman (Card + foto + nama + umur)
- Tombol Greet untuk menampilkan pesan halo
- Tombol Reset untuk menghapus pesan
- Tambah teman baru via input (nama + umur)
- Hapus teman dari list
- Styling responsive + hover effect

## Tools
- ReactJS via CDN
- HTML, CSS
- Live Server (VSCode) untuk menjalankan
